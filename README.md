# Task 1: Business Sales Performance Analytics

## Future Interns - Data Science & Analytics  
**CIN ID:** FIT/MAY26/DS17705  

##  Dataset
- Superstore Sales (9,995 orders)  
- Total Revenue: ₹22,94,505  
- Total Profit: ₹2,85,703  

##  Analysis

### Top Products
Analysis of best-selling products based on sales and profit contribution.

### Regional Performance
Comparison of sales performance across different regions.

### Category Performance
Evaluation of performance across product categories.

### Time Trends
Analysis of sales patterns over time (monthly/seasonal trends).

##  Key Insights
1. Technology category generated the highest revenue  
2. West region was the best performing region  
3. Holiday season (December/January) showed peak sales  

##  Tools Used
- Excel Pivot Tables  

##  Learning Outcomes
This analysis helped me understand:
- How to clean and analyze sales data  
- How to extract business insights from raw numbers  
- How to use pivot tables for quick and effective analysis
